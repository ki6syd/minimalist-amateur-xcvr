Responses

404

post /cw
Send a CW message (cwPost)

Query parameters

messageText (required)
Query Parameter — String to send
speed (optional)
Query Parameter — Keyer speed while sending CW message [wpm]
rfFrequency (optional)
Query Parameter — Dial frequency for sending [Hz]. If nothing provided, radio will decide the best frequency using its own logic. Radio may use the existing dial frequency, beacon frequency, etc.

Responses

201

Message added to queue

400

Invalid / not understood

409

Unable to accept message into queue

Up 
post /ft8
Send a FT8 free text message (ft8Post)

Query parameters

messageText (required)
Query Parameter — 13 character free-text FT8 message to send
timeNow (optional)
Query Parameter — Caller's current time [epoch ms]. If provided, radio will update its current time to match timeNow. If not provided, radio will use its best estimate of current time.
rfFrequency (optional)
Query Parameter — Dial frequency for FT8 [Hz]. If nothing provided, radio will decide the best frequency using its own logic. Radio may use standard digital mode frequency in the current current band, a pre-set configured frequency for sending all FT8 messages, etc.
audioFrequency (optional)
Query Parameter — Audio frequency (USB) relative to RF dial frequency [Hz]. Radio will pick an audio frequency if nothing provided. May be randomized, fixed 1500 Hz, configured in settings file, etc.

Responses

201

Message added to queue

400

Invalid / not understood

409

Unable to accept message into queue
Up
